## Instaling Bot

Complete all your bot info in config.json
```
* Token = Your_Bot_Token
* Prefix = your_BOT_Prefix
* Owner = your_Discord_id
```
- You can find Your_Bot_Token [right here](https://discordapp.com/developers)
